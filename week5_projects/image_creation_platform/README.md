# 智能图像创作平台

## 项目概述
构建一个完整的AI图像创作平台，支持文本生成图像、图像编辑和风格迁移等功能。

## 项目结构
```
image_creation_platform/
├── backend/
│   ├── api/              # FastAPI接口定义
│   ├── models/           # 模型实现
│   ├── services/         # 业务逻辑
│   └── utils/            # 工具函数
├── frontend/
│   ├── components/       # UI组件
│   ├── pages/           # 页面
│   └── services/        # API调用
└── deployment/
    ├── docker/          # 容器配置
    └── k8s/             # K8s配置
```
